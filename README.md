# Telecom Customer Churn Prediction

##  Overview
This project predicts whether a telecom customer is likely to churn based on customer behavior and service usage.

##  Features
- End-to-end machine learning pipeline
- Real-time churn probability prediction
- Deployed ML model using FastAPI

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- FastAPI

##  Use Case
Helps telecom companies proactively identify high-risk customers and improve retention strategies.
